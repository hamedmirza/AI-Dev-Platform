# Runbook

## Before coding
- Start LM Studio local server if using local model backend.
- Confirm the configured model is loaded.
- Enable Cursor project rules.
- Read `docs/MASTER_BUILD_SPEC.md`.

## First execution target
- Complete phase 1 only.
- Validate:
  - app import succeeds
  - settings load
  - health endpoint returns
  - provider health route exists
